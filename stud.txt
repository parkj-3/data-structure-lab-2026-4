깃허브 올리는법
1. 로컬과 원격저장소 연결
▪ $ git remote add origin https://github.com/(your_username)/(Repository_name).git 
2. 잘 연결되었는지 확인
▪ $ git remote -v
3.로컬 저장소의 커밋을 push 명령어로 원격저장소에 올리기
▪ $ git push –u origin master
4. 원격저장소에서 내려받기 (Clone)
▪ $git clone https://github.com/parkj-3/data-structure-lab-2026-4.git .
▪ 내려받기 확인
5. 다른 작업 폴더에서 다시 내려받기
▪ $ git pull origin master
6. 로컬저장소에서 수정 후 올리기
▪ $git add .
▪ $git commit –m “hi”
▪ $git push origin master
7. 수정된 사항 다시 내려받기
▪ 다른 작업 폴더에서 다시 내려받기
▪ $ git pull origin master
